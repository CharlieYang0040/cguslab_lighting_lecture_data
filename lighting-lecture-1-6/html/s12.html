노출 조절 실습 가이드

최적의 노출 설정을 찾는 단계별 접근법

단계별 노출 조절 워크플로우

1기본 설정 초기화

표준 ISO(100-400), f/8, 1/60초 설정으로 시작

2테스트 렌더링

낮은 해상도로 빠른 테스트 렌더링 수행

3히스토그램 확인

색상 분포가 한쪽으로 치우치지 않도록 확인

4주요 파라미터 조절

ISO: 전체 밝기 조절
셔터 속도: 모션 블러 고려
조리개(F값): 피사계 심도 고려

5세부 조정

하이라이트와 그림자 디테일을 확인하며 미세 조정

실습 팁

  • 파라미터 변경 시 한 번에 하나씩 조절하며 영향 관찰
  • 실내 장면: ISO 400-800이 좋은 출발점
  • 외부 장면: ISO 100-200으로 시작
  • 렌더링 시간 단축을 위해 Region Render 활용
  • 조명 설정이 정확해야 사실적 노출 가능

일반적인 문제 해결

📉 이미지가 너무 어두움

• ISO 증가

• 셔터 속도 감소(값 낮추기)

• 조리개 값(F값) 낮추기

📈 이미지가 너무 밝음

• ISO 감소

• 셔터 속도 증가(값 높이기)

• 조리개 값(F값) 높이기

노출 비교 예시

적정 노출

적정 노출
  • 디테일이 적절히 유지됨
  • 하이라이트와 그림자 밸런스
  • 색상 재현 정확

과다 노출

과다 노출
  • 하이라이트 디테일 손실
  • 색상 왜곡

부족 노출

부족 노출
  • 그림자 디테일 손실
  • 노이즈 증가

적정 노출 평가하기

V-Ray에서 노출 확인 방법:

  1. VFB(V-Ray Frame Buffer)의 히스토그램 도구 활용
  2. RGB 값 확인: 하이라이트 255 미만, 그림자 10 이상 유지
  3. 색상 클리핑 표시 기능으로 과다 노출 부분 확인
  4. 최종 판단은 육안 검사로 자연스러움 확인
